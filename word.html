Página Inicial

Explorar a interface

Conheça a interface do Microsoft Word.

Formatação de Fontes

Você pode alterar a fonte, o tamanho da fonte, a cor e outros atributos de texto.

Use os botões Negrito, Itálico e Sublinhado para formatar o texto conforme necessário.

Inserir

Clique no botão Imagem para inserir uma imagem do seu computador ou de um local online.

Você pode redimensionar, mover e formatar a imagem conforme necessário.

Tabelas:

Use o botão Tabela para criar uma tabela.

Escolha o número de linhas e colunas e personalize a aparência da tabela.

Gráficos:

O botão Gráfico permite criar gráficos e representações visuais dos seus dados.

Selecione o tipo de gráfico (como gráfico de barras, pizza ou linha) e insira seus dados.

Hiperlinks:

Insira links para páginas da web, arquivos locais ou endereços de email.

Selecione o texto ou imagem que deseja transformar em um link e clique no botão Hiperlink.

Cabeçalho e Rodapés:

Use os botões Cabeçalho e Rodapé para adicionar informações consistentes em todas as páginas do seu documento.

Por exemplo, você pode incluir o número da página, o título do documento ou outras informações relevantes.

Símbolos e Equações:

O botão Símbolo permite inserir caracteres especiais, como símbolos matemáticos ou letras acentuadas.

Use o botão Equação para criar fórmulas matemáticas complexas.

Comentários:

Use o botão Comentário para adicionar notas ou feedback ao seu documento.

Isso é útil para colaboração ou revisão.

Formatação

Tema: Permite escolher um tema para o documento, que inclui fontes, cores e efeitos de formatação.

Configurar Página: Oferece opções para configurar o tamanho do papel, as margens, a orientação da página e a quebra de texto.

Parágrafo: Permite ajustar a indentação, o espaçamento entre linhas e parágrafos, e o alinhamento do texto.

Organizar: Oferece opções para posicionar e organizar elementos gráficos no documento.

Quebra: Permite inserir quebras de página, seção, coluna e texto.

Plano de Fundo: Oferece opções para adicionar uma cor de fundo, uma marca d'água ou uma imagem ao documento.

Exibir

Modos de Exibição: Oferece opções para alternar entre diferentes modos de exibição, como Layout de Impressão, Modo de Leitura e Modo de Estrutura de Tópicos.

Zoom: Permite ajustar o zoom do documento.

Janela: Oferece opções para dividir a janela do documento ou alternar entre diferentes documentos abertos.

Referências

Índice: Permite criar e gerenciar um índice para o documento.

Notas de Rodapé: Permite adicionar e formatar notas de rodapé.

Citações e Bibliografia: Facilita a inserção de citações e referências bibliográficas.

Legendas: Permite adicionar legendas a figuras, tabelas e outros elementos.

Índice Remissivo: Permite criar e gerenciar um índice remissivo para o documento.

Tabelas e Figuras: Oferece opções para gerenciar tabelas e figuras no documento.

Revisar

Revisão de Texto: Oferece opções para verificar a ortografia, a gramática e o estilo do texto.

Controle de Alterações: Permite controlar as alterações feitas no documento por diferentes autores.

Comentários: Permite adicionar e gerenciar comentários no documento.

Comparar: Oferece opções para comparar duas versões diferentes do documento.

Proteger: Permite proteger o documento com senha ou restrições de edição.
